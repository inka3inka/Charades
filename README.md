## General info
This project is a simple game for two teams based on guessing passwords.

## Technologies
Project is created with:
* React
* React Context
* SASS
* webpack

## Features
* Enter and remember teams' names (with validation)
* Fetch passwords
* Draw passwords (random)
* Mark result
* Calculate score
* Reset game
* State management
